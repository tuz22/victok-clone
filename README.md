#### 0. 프로젝트명
```
- victok-clone
```

#### 1. 주제
```
- victok 페이지를 클론한 리액트 프로젝트
```


#### 2. 구현 기능

```
- pages
    - 메인 페이지
    - 로그인 페이지
    - 회원가입 페이지(이용약관/회원가입/기본정보/시설정보/회원가입완료)
    - 비밀번호 재설정 페이지
    - 라카현황(데이터 있을 때/없을 때)
- components
    - 헤더 / 푸터
    - 로그인
    - 회원가입
        - 정규표현식
        - 우편번호API
        - 체크박스 전체선택/해제
    - 라카현황 JSON 데이터 map으로 작성
- redux로 회원정보 관리
- router로 페이지 전환 관리
```
